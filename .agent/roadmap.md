# figure-verification ‚Äî roadmap

Local "verified-plot" PoC. A weak local LLM only PROPOSES a restricted JSON chart spec (VPlot); a separate trusted verifier deterministically recomputes the plotted data from the source CSV, runs structured checks, blocks charts whose spec, encoding, policy, or dataset binding fail those checks, and renders only verified charts with a provenance certificate (dataset hash, spec hash, plotted-table hash, passed checks).

- **Scope-seed**: `.agent/outline.md` ‚Äî the original outline as 16 verbatim seed steps "Milestone 0..15" (commit `9d09ecb`). The ledger below maps each routine-milestone `M<m>` to those steps; read the relevant seed step on demand when planning a milestone.
- **Stack + determinism invariants**: `.agent/memory.md` (Stack / Determinism sections) ‚Äî researched SOTA, deliberately overriding the outline's human-popular defaults.
- **Data-flow (trust spine)**: the untrusted model proposes ONLY a VPlot spec (transforms + encoding + declared `dataset.hash`) ‚Äî never plotted values. The verifier recomputes ALL plotted data; the renderer inlines only that. So lies needing model-supplied data (the seed's "plots a value ‚âÝ recomputation") are impossible by construction, not checks; checks target spec/encoding/policy/dataset-binding consistency. (The seed's `aggregates_match_recomputation` example carries a model-supplied value ‚Äî a seed inconsistency, resolved here.)
- **Modest claim** (hold the line): verified = {validated spec, the independently recomputed plotted table, the emitted Vega-Lite inlining only that table, the provenance badge} are mutually consistent and the checks passed. Trusted, NOT verified (TCB): `vl-convert`/Vega, SVG rasterization, browser, pixels ‚Äî trusted to render verified data faithfully, not proven to.
- **Quality gate** (M1.1 wires it; every WORK-UNIT VERIFY runs it, all green, touched scripts exit clean): `ruff format --check .` ¬∑ `ruff check .` ¬∑ `mypy` ¬∑ `pytest` ‚Äî all via `uv run --locked` (the lockfile, not a newer floor-satisfying release, pins the gate).

Arrays are bounded tuples, not lists (finding 7 ‚Äî deep immutability + hashable specs). One module-level `Decoder(VPlotSpec)`. `decode_spec(raw)` = decode, then a stdlib `json.loads(object_pairs_hook=‚Ä¶)` dup-key pre-scan raising `ValidationError` (finding 4). `json_schema()` = `{**msgspec.json.schema(VPlotSpec), "$schema": <draft-2020-12 URI>}` ‚Äî `$schema` stamped LAST so a future msgspec emission cannot override it (finding 5: today msgspec emits none); `json_schema_text()` = `json.dumps(‚Ä¶, indent=2, ensure_ascii=False)+"\n"`.

#### M1.4a ‚Äî Canonical forms + provenance hashing (`verifier/canon.py`)
Evaluator-independent foundation (no transform logic; M1.4b/c import it).
- Value model: `Cell = Decimal | str | None` (numeric / temporal+string canonical text / the one null token); `Column`(name, `kind`‚àànumeric|temporal|string, `scale` for numeric, `granularity` for temporal ‚Äî NO unit/label: display metadata stays in the manifest so it never enters the data hash); `Table`(columns, rows already in canonical total order ‚Äî M1.4b's closure produces them).
- `serialize_table` ‚Üí typed-NDJSON: a header JSON array of self-describing `name:numeric:<scale>` / `name:temporal:<granularity>` / `name:string` descriptors, then one compact JSON row-array each (`ensure_ascii=False`, `separators=(",",":")`), numerics quantized to exactly `scale` places HALF_EVEN, null ‚Üí JSON `null` (unambiguous vs the string `"null"`), trailing `\n`.
- Hashes (SHA-256 over text, never Arrow/Parquet): `hash_dataset(csv_bytes)` = raw bytes, `sha256:`-prefixed, NO domain tag (‚â° the spec's declared `dataset.hash` source identity; sensitive to row-order/CRLF/BOM by design); `hash_table`/`hash_spec`/`hash_manifest` via `_digest(domain, payload)` = sha256 of a `vplot-<domain>/<CANON_VERSION>\n` tag + payload (a serialization-format bump invalidates stale tagged hashes; the raw dataset hash stays format-free). `hash_spec` = `Encoder(order="deterministic").encode(spec)` (finding 8: definition-order fields, Decimal‚Üístring, no Unicode pass) ‚Üí NFC-normalize the decoded text ‚Üí digest (NFC of the whole encoded JSON ‚â° NFC of its string values; structural chars are NFC-stable). `hash_manifest` = digest of the raw manifest bytes.
- `CANON_VERSION` tag + `EXPECTED_UNIDATA="15.1.0"`; `hash_spec` asserts `unicodedata.unidata_version == EXPECTED_UNIDATA` (RuntimeError otherwise, so a UCD shift cannot silently move spec hashes). `runtime_versions() ‚Üí Versions`(canon_version, python, msgspec, unidata) ‚Äî the M1.6 badge stamps these.
- `tests/test_canon.py`: dataset hash flips on any source byte change (row-order/CRLF/BOM), stable on identity; table hash flips on cell/column/scale change, equal for tables that serialize identically; spec hash flips on a spec edit, EQUAL for NFC-equivalent strings (compose vs decompose a combining mark), stable across re-encode; manifest hash flips on byte change; `runtime_versions` populated; the unidata guard fires via monkeypatch; all four hashes byte-identical across two `PYTHONHASHSEED` subprocesses (memory Ops: seed varies only via subprocess).
- **Accept**: dataset hash = tag-free raw-byte identity (order-sensitive); table/spec/manifest domain-tagged; spec hash NFC-stable + asserts the pinned UCD; every hash deterministic across PYTHONHASHSEED subprocesses; gate green at 100% branch coverage.

#### M1.4b ‚Äî Deterministic evaluator (`verifier/eval.py`)
Imports M1.4a (`Cell`/`Column`/`Table`). Hand-rolled, Decimal-exact (memory invariants: no float, no Kahan), implementing `VPlot_SEMANTICS.md` ¬ß2‚Äì6:
- Manifest: a msgspec tagged union (on `type`) mirroring `data/schemas/<name>.json` ‚Äî numeric{name, scale‚â•0, unit?, label?} / temporal{granularity} / string; `Manifest`{dataset, columns}; `load_manifest` strict-decodes; each ‚Üí a `canon.Column`.
- Typed load `load_table(csv_bytes, manifest)`: UTF-8 + `csv.reader`; header must equal the manifest column names; per cell empty‚ÜíNone, else coerce by type ‚Äî numeric‚Üí`Decimal`, finite, ‚â§ `scale` decimal places (else error); temporal‚Üí`date`/`datetime.fromisoformat` with a re-`isoformat()`==input canonicality check; string verbatim (`NA` stays a string ‚Äî memory M1.3). Row length must match.
- Pipeline (match on `op`, ¬ß4): `select`=projection no-dedup; `filter`=3-valued, null/incomparable‚Üídrop, the value coerced to the field type per ¬ß3 (int-vs-temporal/string mismatch‚Üíerror); `group_by`=null key its own group, valid only immediately before `aggregate`; `aggregate`= count(non-null integer, scale 0) / sum,mean(numeric input only; mean = exact Œ£ √∑ non-null count, ONE division, quantize HALF_EVEN to source scale) / min,max(inherit source) per group, zero non-nulls‚Üínull, no preceding group_by‚Üíwhole-table single row. Distinctness ¬ß5.
- Total-sort closure ¬ß6: active sort = the LAST `sort` op (reset by an intervening `aggregate`); final table sorted by active keys then all remaining columns ascending, null greatest ‚Üí permutation-invariant. Sort fields must exist in the final plotted schema.
- `evaluate(spec, manifest, csv_bytes) ‚Üí Table` is the sole public entry; the plotted schema is `Table.columns` (no separate `plan_columns` ‚Äî M1.5 derives schema from this). Semantic failures raise `SemanticError(msg, *, check)`; eval-layer `check` machine-names: `schema.fields_exist`, `schema.field_types_match`, `filter.value_type`, `transform.group_by_placement`, `aggregate.output_unique`, `sort.fields_distinct`, `sort.field_in_plotted_table` (+ select/group_by distinctness).
- `tests/test_eval.py`: each good spec ‚Üí its canonical plotted table, locked golden (syrupy) AND hand-verified at creation against the semantics, with representative cases also asserted INLINE (sum/mean HALF_EVEN rounding, empty-group‚Üínull, total-sort nulls-greatest, whole-table aggregate) so the lock is not self-blessing; a semantically no-op spec edit (a sort superseded by a later one) yields an identical plotted-table hash; the eval-layer bad specs (`decodes=true` fixtures whose `caught_by` includes M1.4 ‚Äî b07/b09/b10/b14/b15/b16) each raise `SemanticError` with their `check` (encoding/label/binding fixtures b08/b11/b12/b13 stay M1.5 ‚Äî eval never inspects encoding); `deliberately_dirty.csv` drives the coercion-error branches; inline-constructed specs cover semantic branches the corpus does not exercise (select/group_by distinctness, sort-field-absent-from-table, an all-null group ‚Üí empty aggregation, filter ¬ß3 coercion rows beyond int-vs-string).
- **Accept**: good specs ‚Üí row-for-row reproducible golden tables (semantics-verified, not blindly snapshotted); a no-op spec edit leaves the plotted-table hash fixed (spec edits move only the spec hash); each eval-layer bad spec raises its specific `check`; aggregation Decimal-exact (no float); gate green at 100% branch coverage.
